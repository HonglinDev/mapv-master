# 文档自定义样式示例

欢迎来到文档自定义样式示例页面！本页面展示了完整的文档样式配置，您可以通过右侧的样式自定义组件实时调整文档样式。

<StyleCustomizer />

## 样式自定义组件说明

样式自定义组件提供了以下功能：

### 1. 字体样式调整
- **字体类型**：支持多种字体族选择
- **字体大小**：12px 到 24px 可调节
- **字体粗细**：300 到 700 可选择
- **行高**：1.0 到 3.0 可调节

### 2. 颜色方案配置
- **文本颜色**：支持颜色选择器和十六进制颜色码输入
- **背景颜色**：支持颜色选择器和十六进制颜色码输入
- **主题色**：自定义文档的主题颜色

### 3. 间距设置
- **段落间距**：8px 到 48px 可调节
- **标题间距**：16px 到 64px 可调节
- **内边距**：8px 到 48px 可调节

### 4. 布局调整
- **对齐方式**：左对齐、居中、右对齐、两端对齐
- **最大宽度**：600px 到 1400px 可调节

### 5. 样式预设管理
- **保存预设**：将当前样式保存为预设
- **加载预设**：加载已保存的样式预设
- **删除预设**：删除不需要的样式预设
- **重置样式**：恢复为默认样式

## 如何使用

1. 在左侧样式控制面板中调整各项样式参数
2. 右侧实时预览区域会立即显示样式更改效果
3. 点击"保存预设"按钮可将当前样式保存为预设
4. 在"已保存的预设"列表中可以加载或删除预设
5. 点击"重置为默认样式"按钮可恢复为初始样式

## 样式应用范围

自定义的样式会应用到预览区域中的所有内容，包括：
- 标题（h1-h6）
- 段落文本
- 列表（有序列表和无序列表）
- 表格
- 代码块和行内代码
- 引用
- 链接
- 分隔线

## 技术实现

本样式自定义组件基于 Vue 3 开发，使用了 CSS 变量实现样式的动态调整，通过 localStorage 实现样式预设的持久化存储。组件采用了响应式设计，在不同屏幕尺寸下都能保持良好的使用体验。
