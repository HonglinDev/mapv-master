# Examples文件夹改造计划

## 1. 需求分析

* 将examples文件夹中的HTML案例集成到VitePress文档系统中

* 设计案例展示功能，确保HTML文件能被正确解析和渲染

* 添加导航结构，方便用户浏览不同HTML案例

* 优化视觉效果，与文档整体风格保持一致

* 确保代码可阅读性，添加代码高亮和复制功能

* 实现响应式设计，适配不同设备

## 2. 实施步骤

### 2.1 设计案例展示组件

* 创建自定义Vue组件`ExampleViewer.vue`，用于展示HTML案例

* 组件功能：

  * 使用iframe嵌入HTML案例

  * 显示案例标题和描述

  * 展示案例源代码，支持代码高亮和复制

  * 响应式设计，适配不同屏幕尺寸

### 2.2 组织案例结构

* 按平台和可视化类型重新组织案例

* 为每个案例生成对应的Markdown文件

* Markdown文件包含：

  * 案例标题和描述

  * 使用`ExampleViewer`组件嵌入HTML案例

  * 案例源代码展示

### 2.3 优化导航结构

* 修改`config.mjs`，更新侧边栏配置

* 按平台（百度地图、高德地图、Leaflet等）分类案例

* 每个平台下按可视化类型（点图、热力图、网格图等）进一步分类

### 2.4 实现代码高亮和复制功能

* 使用VitePress内置的代码高亮功能

* 添加复制按钮，支持一键复制代码

* 优化代码展示样式，提高可读性

### 2.5 响应式设计优化

* 确保iframe在不同屏幕尺寸下自适应

* 调整布局，在小屏幕上优化显示效果

* 确保导航结构在移动端友好

## 3. 技术实现

### 3.1 自定义组件实现

```vue
<template>
  <div class="example-viewer">
    <h2>{{ title }}</h2>
    <p>{{ description }}</p>
    <div class="example-container">
      <iframe :src="src" frameborder="0" class="example-iframe"></iframe>
    </div>
    <div class="code-container">
      <h3>源代码</h3>
      <div class="code-header">
        <button @click="copyCode" class="copy-btn">复制代码</button>
      </div>
      <pre><code :class="language">{{ code }}</code></pre>
    </div>
  </div>
</template>
```

### 3.2 案例生成脚本

* 编写Node.js脚本，自动扫描examples文件夹

* 为每个HTML案例生成对应的Markdown文件

* 提取案例标题和描述

* 生成组件使用代码

## 4. 预期效果

* 所有HTML案例都能在文档系统中正确展示

* 提供清晰的导航结构，方便用户浏览

* 案例展示与文档整体风格一致

* 代码可阅读性高，支持一键复制

* 响应式设计，适配不同设备

* 组织案例结构和生成Markdown文件：1天

* 优化导航结构：半天

* 实现代码高亮和复制功能：半天

* 响应式设计优化：半天

## 6. 风险评估

* HTML案例中可能包含外部资源引用，需要确保资源路径正确

* iframe嵌入可能导致跨域问题，需要处理

* 大量案例可能导致构建时间过长，需要优化

## 7. 解决方案

* 使用相对路径引用资源，确保资源能正确加载

* 配置VitePress允许iframe嵌入

* 优化构建配置，提高构建效率

## 8. 验收标准

* 导航结构清晰，方便用户浏览

* 视觉效果与文档整体风格一致

* 代码高亮和复制功能正常工作

* 响应式设计适配不同设备
